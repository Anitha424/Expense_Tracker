# Frontend (React + Vite + Tailwind)

Expense Tracker UI with authentication, dashboard, and analytics.

## Pages

- Login Page
- Register Page
- Dashboard Page
- Analytics Page

## Components

- Navbar
- AddTransactionForm
- TransactionList
- SummaryCards
- AnalyticsCharts

## Run Locally

1. Copy `.env.example` to `.env`
2. Install dependencies:
	- `npm install`
3. Start dev server:
	- `npm run dev`

## Environment

- `VITE_API_URL`: backend API base URL (default: `http://localhost:5000/api`)
